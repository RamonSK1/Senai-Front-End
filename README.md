# 🎮 Axion games

Projeto desenvolvido durante o curso de **Codificação Front-End** do **SENAI**, utilizando HTML, CSS, JavaScript, Bootstrap e jQuery.

## 📖 Sobre o projeto

A **Axion Games** é uma página web de uma loja virtual de jogos. O objetivo do projeto é praticar conceitos de desenvolvimento front-end, criando uma interface moderna, responsiva e interativa.

O sistema possui páginas de navegação, exibição de produtos e uma tela de login e cadastro com validação em JavaScript.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- jQuery

## ✨ Funcionalidades

- Página inicial da loja
- Barra de navegação responsiva
- Busca de produtos
- Cards de jogos
- Carrossel de destaques
- Página de login
- Página de cadastro
- Botão "Voltar ao topo"
- Validação de login utilizando JavaScript
- Interações com jQuery



## ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/RamonSK1/Senai-Front-End.git
```

2. Entre na pasta do projeto:

```bash
cd Senai-Front-End
```

3. Abra o arquivo `index.html` em seu navegador.

Não é necessário instalar dependências ou executar servidor.
